## СЛАЙД 2: АКТУАЛЬНІСТЬ ПРОБЛЕМИ

### Чому це важливо?

**Проблема 1: Лісові пожежі**
- 1,875 людських пожеж у лісових масивах України
- 80% рекреаційних пожеж відбуваються ПОЗА офіційними місцями
- Економічні збитки та екологічна шкода

**Проблема 2: Стихійний туризм**
- Неконтрольований вплив на об'єкти ПЗФ
- Відсутність інфраструктури біля природних атракторів
- Нераціональне використання рекреаційного потенціалу

**Проблема 3: Відсутність наукового підходу**
- Інвестиційні рішення базуються на інтуїції
- Немає системного аналізу територій
- Брак геопросторової візуалізації даних

### Рішення: ГІС для науково обґрунтованого вибору локацій

## СЛАЙД 5: НАУКОВО-МЕТОДОЛОГІЧНА БАЗА

### Метод аналітичної ієрархії (AHP)

**Автор:** Томас Сааті (1980)

**Принцип:** Парне порівняння критеріїв для визначення відносної важливості

**Матриця переваг (фрагмент):**

|          | Попит | ПЗФ | Природа | ... |
|----------|-------|-----|---------|-----|
| Попит    | 1     | 2   | 3       | ... |
| ПЗФ      | 0.5   | 1   | 2       | ... |
| Природа  | 0.33  | 0.5 | 1       | ... |

**Результат:**
- Попит: 25% (25 балів)
- ПЗФ атрактор: 20% (20 балів)
- Природа: 15% (15 балів)
- Транспорт: 15% (15 балів)
- Інфраструктура: 10% (10 балів)
- Пожежі: 5% (5 балів)
- Насиченість: 15% (штраф 0 до -15)

**Перевірка узгодженості:**
- **CR = 0.16% < 10%** (відмінна узгодженість)

### Наукові джерела

- Saaty T.L. (1980) - базова методологія AHP
- Kentucky SCORP 2020-2025 - стандарти рекреаційного планування
- NW Fire Science Consortium - дослідження профілактики пожеж
- Закон України "Про природно-заповідний фонд"

## СЛАЙД 6: 7 ФАКТОРІВ ОЦІНКИ

### Інтегральний показник рекреаційного потенціалу

**Формула:**
```
P = F₁ + F₂ + F₃ + F₄ + F₅ + F₆ + F₇
Максимум: 90 балів
```

### Фактори з вагами:

**1. ПОПИТ ВІД НАСЕЛЕННЯ (25%)**
- Розрахунок річного попиту на рекреацію
- Оцінка існуючої пропозиції
- Визначення дефіциту/профіциту

**2. ПЗФ ЯК АТРАКТОР (20%)**
- Зважена оцінка об'єктів (НПП ×5, РЛП ×3)
- Коригування на площу області

**3. ПРИРОДНІ УМОВИ (15%)**
- Лісове покриття (0-11 балів)
- Водні об'єкти (+4 бали)

**4. ТРАНСПОРТНА ДОСТУПНІСТЬ (15%)**
- Міжнародні траси: 15 балів
- Національні: 12 балів
- Регіональні: 8 балів

**5. ІНФРАСТРУКТУРА (10%)**
- Готелі у столиці області
- Існуючі рекреаційні об'єкти

**6. ПРОФІЛАКТИКА ПОЖЕЖ (5%)**
- Бонус за наявність людських пожеж
- 10+ пожеж = +5 балів

**7. НАСИЧЕНІСТЬ (-15%)**
- Штраф за конкуренцію у радіусі 5 км
- 0 конкурентів: 0, 13+: -15 балів

## СЛАЙД 9: АЛГОРИТМ ІДЕНТИФІКАЦІЇ ЗОН

### 3 категорії рекомендованих локацій

**1. Кластери людських лісових пожеж**

**Алгоритм:**
```python
for fire in human_fires:
    nearby = count_fires_within(fire, radius=10km)
    if nearby >= 3:
        cluster_center = calculate_centroid(nearby)
        zones.append(cluster_center)
```

**Обґрунтування:**
- 80% рекреаційних пожеж поза офіційними місцями (NW Fire Science)
- Облаштовані місця знижують ризик на 40%

**2. Зони біля об'єктів ПЗФ**

**Алгоритм:**
```python
top_pfz = select_top_2_by_visitors(region)
for pfz in top_pfz:
    distance = random(3, 8)  # км
    zone = offset_from_center(pfz, distance)
    zones.append(zone)
```

**Обґрунтування:**
- Близькість до атрактора (3-8 км)
- Мінімізація екологічного впливу

**3. Зони вздовж міжнародних трас**

**Алгоритм:**
```python
for road in international_roads:
    points = generate_along_road(road, interval=50-100km)
    zones.extend(points)
```

**Обґрунтування:**
- Потреба транзитних туристів у зупинках
- Інтервал 50-100 км = середня відстань для відпочинку

### Радіус зони будівництва: ±5 км

У межах 5 км від центральної точки:
- Перевірка конкуренції
- Пошук конкретного місця
- Оцінка локальних умов
